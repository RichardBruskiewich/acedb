Physical_map_calculation

This program tries to analyse the clone map, also called physical map It is quite primitive and not linked in distribution 1.4.

Menu items:

  • Quit: quits the window.
  • Help: brings up this help.
  • Print: brings up the print window.
Make pMaps should be used to recalculate the contigs after updating the clones.

Align is itself a subprogram.

The other buttons as a whole make a program and should be used in sequence.
