TableMaker: Data Window

Displays results for applying a TableMaker definition.

Menu items and Buttons in the Data Window

  • Edit: Click on this button to invoke editing mode (button turns blue). Then double-click any item in the data display if you want to change its value. If you pick an object, you will be alerted if you attempt to enter a name that is not recognized. Warning: if you have taken write access you can modify data in the database. Otherwise you alter only the data display.
  • Quit: destroys the window.
  • Help: displays this help page.
  • Print: prepares a Post Script file in $ACEDB/PS which you can print to your local laser printer.
  • Switch: First touch an element in some column, then press Switch, this will exchange the active column with the one immediately to its right. At the same time, the lines are reordered so that they always reamain in alphanumerical order left to right. By Putting the genes left of their position, you get them in alphabetic order, by putting first the chromosome, then the position, then the gene, you get them in genetic map order.
  • Export KeySet: Exports the active column to a new KeySet window if it contains keys.
  • Define Table: Returns to the TableMaker definition window.
  • Map: Construct a MultiMap based on the visible numerical columns.
  • Ascii Dump: Dump the table in Ascii format. You can choose any single character as a column delimiter.
  • Import Active Keyset: will apply the table definition to the active KeySet and display the result. This overwrites the original contents in the data display window.
  • Reset Table Display: redraws display.
