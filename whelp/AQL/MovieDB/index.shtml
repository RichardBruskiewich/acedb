AQL - MovieDB Tutorial

In order top demonstrate some basic language capabilities of AQL, a small example database has been designed.

This database of movie-related data shows the use of ACeDB for non-biological data and provides an easy playground to get used to AQL.

The data has mainly been extracted from the Internet Movie database.
